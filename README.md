# Description

Simple portfolio website built with React and Tailwind CSS for styling. 

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Features

- React Smooth Scroll
- Tile displays for tech stack and project directories
- Hover effects on tile display
- A Contact form synced directly to my email and hosted by getform.io
- Mobile responsive design
- Interactive hamburger menu

## Tech Stach

- React
- TailwindCSS
- getform.io
- yarn

## Execution

- Clone repository
- Run yarn install
- Execute the start command

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
